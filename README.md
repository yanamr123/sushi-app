# sushi-mulatorとは
コロナ禍で回転寿司に行けない人のための、家で寿司を注文した気になれるwebアプリです。  
UIは個人的に大好きなスシローさんの注文UIを参考にして作りました。
  
### URL
Firebaseにデプロイ中です  
URL:  <https://sushimulator.web.app/>
  
### 感想
M1の夏にHTML,CSS,JavaScriptの勉強を始めて10日程で作りました。
フレームワークを勉強しようと思いました。
